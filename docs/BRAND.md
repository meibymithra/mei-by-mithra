# Brand Website Notes

## Positioning

`Mei by Mithra` is the public-facing brand website for `Mithra Krishnamoorthy`.

The website should feel like:

- a brand
- a portfolio
- a trust surface
- a service entry point
- a digital product storefront

It should not feel like:

- a generic therapy template
- a SaaS dashboard with a public skin
- a plain resume site

## Visual direction

- Editorial spacing
- Warm earth palette
- Intentional motion
- Portrait-led storytelling
- Soft but confident surfaces
- Organic linework and atmospheric backgrounds

## Typography

- Heading: `Bantayog`
- Body: `Playpen Sans`

## UI principles

- The first screen should establish Mithra, not abstract service language
- Homepage should establish brand and route visitors into dedicated pages
- Portfolio depth should sit on a dedicated page, not inside the primary homepage narrative
- Booking and store actions should both be visible
- The site should work equally for local users and Indians abroad
- Forms should feel calm and guided, not bureaucratic
- Store content should explain audience and use, not just product names

## Messaging pillars

- Rights
- Respect
- Responsibility
- Educate

## Public conversion paths

1. Book a session
2. Complete intake after email
3. Receive invoice / payment link
4. Complete session
5. Submit feedback
6. Optional testimonial moderation

Secondary path:

1. Discover a playbook
2. Review audience fit and FAQ
3. Purchase by link or invoice workflow
4. Receive resource delivery
